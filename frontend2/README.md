# frontend2

Fresh Next.js + React + shadcn-style starter for the Detour dashboard UI.

## Stack
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS v4
- shadcn-style component setup (`components.json`, `lib/utils.ts`, `components/ui/*`)

## Current scope
- Header-only UI shell
- `de.png` logo
- Risk pill
- Left-aligned `Last updated` text (not in a pill)
- Black/white theme matching the existing frontend palette

## Run
```bash
cd frontend2
npm install
npm run dev
```

Open `http://localhost:3000`.
